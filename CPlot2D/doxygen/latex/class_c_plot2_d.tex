\subsection{Detailed Description}
The class responsible for storing and rendering all the data sets it contains. Although the currently implemented method for rendering of a plot is into a Post\-Script file, it is envisioned that other rendering methods, such as Open\-G\-L can be implemented as well. The accessors are created for both setting and retrieving plot parameters.
